## 介绍

打开两种图片进行比较，可以通过水平和垂直方向滑动比较预览，也可以按照指定颜色显示比较结果

![](../../assets/images/ToolsSet/TSMImgCompare.png)

## 使用方法

* 上方区域点击可以打开对话框选择源图片和目标图片
* 下方分别为水平和垂直比较预览
* 顶部工具栏可以设置比较结果中相同区域、不同区域和背景的颜色
* 点击工具栏右侧的比较按钮可以打开结果查看界面
