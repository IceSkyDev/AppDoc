## 介绍

功能包括：Encode & Decode、Encrypt & Decrypt & Hash、Text Converter、Text Generator、Text Compare、JSON Formatter、XML Tool、Regex Test、Qr & Bar code、TTS & Morse、Csv Editor、Text Batch、Split Join、List Operate

## 功能列表

### Encode & Decode *编码解码*
* 常用编码转换（11种）
  * Byte数组
  * ASCII
  * UTF-8
  * Iso 8859-1
  * UTF-16
  * UTF-16 BE
  * UTF-32
  * UTF-32 BE
  * GB2312
  * BIG5
  * Shift-JIS
* Base编码转换（13种）
  * Base64
  * Base32（6种）
  * Base58（3种）
  * Base85（3种）

### Encrypt & Decrypt & Hash *加密解密*
* 加解密
  * AES加解密
  * DES加解密
  * RSA加解密
* Hash
  * MD5-16
  * MD5-32
  * SHA1
  * SHA256
  * SHA384
  * SHA512

### Text Converter *文本转换*
* 字母转换
  * 转为大写
  * 转为小写
  * 下划线连接
  * 首字母大写
  * 驼峰模式
  * 中划线连接
  * 常量模式
  * 标题模式
  * 大小写反转
* 中文转换
  * 转为简体
  * 转为繁体
  * 转为无声调拼音
  * 转为带声调拼音
  * 转为拼音标题
  * 转为姓名拼音
  * 转为拼音首字母
  * 拼音转为汉字
  * 拼音转为姓名汉字
* 日文转换
  * 平假名/片假名 -> 罗马字
  * 罗马字/片假名 -> 平假名
  * 罗马字/平假名 -> 片假名
  * 半角字符 -> 全角字符
  * 全角字符 -> 半角字符
  
### Text Generator *文本生成器*
* 密码生成
  * 指定长度
  * 包含数字
  * 包含大小写字母
  * 包含特殊符号
  * 包含自定义字符
* 假文本生成
  * 12种语言（部分类型支持）
  * 34种数据类型
  * 指定数量
  * 指定范围

### Text Compare *文本比较*
* 高亮显示文本比较结果

### JSON Formatter *JSON格式化*
* JSON格式化
* JSON压缩
* JSON扁平化
* JSON反扁平化
* JSON转XML

### XML Tool *XML工具*
* XML格式化
* XML压缩
* XML转JSON
* XPath查找

### Regex Test *正则表达式测试*
* 82种预定义正则表达式
* 正则表达式测试
  * 全局、多行、区分大小写开关
  * 匹配结果分组
  * 显示捕获

### Qr & Bar code *条形码和二维码编码解码*
* 生成条形码或二维码（13种）
* 识别条形码或二维码

### TTS & Morse *文本转语音和摩斯码*
* 文本转语言
  * 直接播放
  * 导出音频
* 摩斯编码转换
  * 文本转摩斯编码
    * 直接播放
    * 导出音频
  * 摩斯编码转文本

### Csv Editor *Csv编辑器*
* 以表格形式编辑CSV文件
  * 支持添加删除行列
  * 支持同时查看文本

### Text Batch *文本批处理工具*
* 网格形式批处理文本
  * 支持公式
  * 支持模板

### Split Join *文本分割合并*
* 文本分割
  * 按数量分割
  * 按字符分割
  * 指定分隔符
* 文本合并
  * 合并多个空格
  * 合并多个换行
  * 合并多个空白字符
  * 合并多个指定字符
* 多行文本连接
  * 用空格连接
  * 用制表符连接
  * 用指定字符连接
* 文本裁剪
  * 指定位置
  * 指定裁剪字符
* 字数统计
  * 按类型统计字符数量
  * 统计ASCII字符数量
  * 统计数量前100的字符
  * 统计指定文本的数量

### List Operate *列表操作*
* 排序
  * 字符升序、降序
  * 随机排序
  * 长度升序、降序
  * 倒序
  * 是否启用数字格式
* 查找
  * 开始位置
  * 结束位置
  * 包含
  * 长度匹配
* Pad操作
  * 指定位置：左、右
  * 指定字符
  * 指定长度
* 提取
  * 指定方向：左、右
  * 按数量提取
  * 按字符提取
* 单一操作
  * 去除重复项
  * 计数
  * 按字符反序
* 列表运算
  * 获取并集
  * 获取交集
  * 排除列表
